# ZoomInfo 데이터셋 샘플

<h2>1001개 ZoomInfo 회사의 샘플 데이터셋</h2>

![ZoomInfo dataset header](https://github.com/bright-kr/ZoomInfo-dataset-samples/blob/main/zoominfo-datasets.PNG)

1000개가 넘는 회사로 구성된 ZoomInfo 데이터셋 샘플입니다. 데이터셋은 <b>Bright Data API</b>를 사용하여 추출되었습니다.

<h2>데이터셋에 포함된 일부 데이터 포인트:</h2>

* ```id```: 게시물의 고유 식별자
* ```user_posted```: 게시물 소유자의 사용자명
* ```name```: 게시물 소유자의 이름
* ```description```: 게시물의 텍스트 설명
* ```date_posted```: 게시물이 게시된 날짜
* ```photos```: 게시물에 첨부된 사진의 URL
* ```videos```: 게시물에 첨부된 비디오의 URL
* ```url```: 게시물로 연결되는 URL 링크
* ```quoted_post```: 메인 게시물 내 인용된 게시물의 세부 정보
* ```tagged_users```: 게시물에 태그된 프로필 목록
* ```replies```: 게시물이 받은 총 답글 수
* ```recompanies```: 게시물이 받은 총 recompanies 수
* ```likes```: 게시물이 받은 총 좋아요 수
* ```views```: 게시물이 받은 총 조회 수
* ```external_url```: 게시물에 포함된 외부 URL
* ```hashtags```: 게시물에 포함된 해시태그
* ```followers```: 해당 프로필의 팔로워 수
* ```biography```: 게시물 소유자의 소개(bio)
* ```companies_count```: 해당 프로필이 만든 총 companies 수
* ```profile_image_link```: 프로필 이미지의 URL
* ```following```: 사용자가 팔로우하는 프로필 수
* ```is_verified```: 사용자가 인증되었는지 여부를 나타냄(True/False)
* ```quotes```: 게시물이 인용된 총 횟수
* ```bookmarks```: 게시물이 북마크된 총 횟수
* ```parent_post_details```: 해당되는 경우, 상위(부모) 게시물의 세부 정보

그 외에도 더 많은 항목이 포함됩니다.

이는 "ZoomInfo Companies (public data)"에서 파생된 샘플 서브셋이며,
해당 데이터셋에는 <b>2,300,000개 이상의 회사</b>가 포함되어 있습니다.

사용 가능한 데이터셋 파일 형식: <b>JSON, NDJSON, JSON Lines, CSV, or Parquet. Optionally, files can be compressed to .gz</b>.

데이터셋 전달 유형 옵션: <b>Email, API download, Webhook, Amazon S3, Google Cloud storage, Google Cloud PubSub, Microsoft Azure, Snowflake, SFTP</b>.

업데이트 빈도: <b>Once, Daily, Weekly, Monthly, Quarterly, or Custom basis</b>.

추출된 데이터 포인트에 추가로 제공 가능한 데이터 enrichment: <b>요청 기반.</b>

<h2>ZoomInfo 데이터셋의 활용 사례는 무엇입니까?</h2>

<h3>1. 효과적인 시장 분석</h3>
포괄적인 자동차 데이터를 활용하여 시장 트렌드를 모니터링하고, 차량 가용성을 추적하며, 소비자 선호도를 분석합니다. Webmotors의 상세한 차량 리스팅을 활용하여 경쟁 인텔리전스와 전략적 시장 계획을 위한 가치 있는 인사이트를 확보할 수 있습니다.

<h3>2. 리드 생성 및 CRM Enrichment</h3>
Webmotors의 고품질 자동차 리스팅 데이터를 통합하여 리드 생성 및 영업 전략을 강화합니다. 상세한 차량 및 판매자 정보를 통해 CRM을 enrichment하여 파이프라인 개발을 가속하고 타겟팅 정확도를 향상할 수 있습니다.

<h3>3. 시장 성장 평가 및 고가치 기회 식별</h3>
상세한 차량 리스팅 및 시장 데이터에 접근하여 성장 잠재력과 업계 트렌드를 평가합니다. 자동차 딜러십, 마케터, 분석가는 Webmotors의 방대한 데이터를 활용해 전략을 고도화하고 수익성 높은 기회를 식별하여 경쟁 시장에서 우위를 유지할 수 있습니다.
